# Portfolio

This is my personal portfolio based on a (slightly updated by myself) simple monospaced resume theme for Hugo created by [ojroques/hugo-researcher](https://github.com/ojroques/hugo-researcher). The latter was further ported from Jekyll theme [ankitsultana/researcher](https://github.com/ankitsultana/researcher). Netlify configuration file included.
